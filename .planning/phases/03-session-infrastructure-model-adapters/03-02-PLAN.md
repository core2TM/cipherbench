| T-03-02-01 | Information Disclosure | LiteLLMAdapter.__init__ | mitigate | API keys read from env vars by LiteLLM automatically (D-14); never accept or store key values as constructor arguments; test_no_api_key_in_source() asserts no hardcoded key patterns |

| T-03-02-SC | Tampering | pip/uv installs (litellm, tenacity) | mitigate | All packages CLAUDE.md-approved; slopcheck table in RESEARCH.md shows no [SLOP]/[SUS] packages |
