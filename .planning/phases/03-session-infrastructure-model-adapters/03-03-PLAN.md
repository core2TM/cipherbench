       f. `attempt_score = self._engine.score_attempt(probe)` â€” build attempt entry with `probe=probe`, `score=attempt_score.score`, `max_score=attempt_score.max_score`, `is_correct=attempt_score.is_correct`, `raw_response=raw`, `extraction_failed=False`, `attempt_num` = `len(self._session_record["attempts"]) + 1`; append to attempts; call `self._writer.write_checkpoint(self._session_record)`; increment `valid_attempts`
       g. If `attempt_score.is_correct`: break (do not use all 5 attempts if correct)
    5.     6. `outcome = "success" if any(a["is_correct"] for a in self._session_record["attempts"]) else "failure"`
